Set Up GitHub Cloud Sync

Connect to a private GitHub Gist to sync across all your devices for free.
Step 1: Create a private Gist at gist.github.com — any filename is fine.
Step 2: Generate a PAT at GitHub → Settings → Developer Settings → Tokens (classic) with gist scope.
Step 3: Paste both below on every device you want to sync.
